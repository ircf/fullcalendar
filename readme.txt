Display and customize one or many Google calendars.

== Description ==

This is just a Wordpress wrapper for the fullcalendar Open Source project.

== Installation ==

1. Download plugin Full Calendar into /wp-content/plugins/.

2. Activate plugin.

3. Visit Google Calendar and create your calendar.

4. Configure Full Calendar in the WordPress Setup menu.

5. Insert the [fullcalendar] shortcode in your pages.
